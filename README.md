# PSD to PNG Converter

PSDファイルを白黒二値化PNGファイルに変換するスタンドアロンアプリケーションです。
教師データ作成用に最適化されています。

## 機能

### メイン変換タブ
- **ファイル/フォルダ読み込み**: PSDファイルまたはフォルダを選択して読み込み
- **バッチ変換**: 複数のPSDファイルを一括でPNGに変換
- **二値化チェック**: 二値化されていないPSDファイルを検出してログ出力
- **出力先指定**: 変換後のPNGファイルの保存先を指定

### 白黒反転タブ
- **白黒反転**: PNGファイルの白黒を反転
- **プレビュー**: 反転前後のプレビュー表示

## インストール

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python main.py
```

## 技術仕様

- **入力形式**: PSD (Adobe Photoshop)
- **出力形式**: PNG (白黒二値化)
- **GUI**: tkinter
- **PSD読み込み**: psd-tools
- **画像処理**: Pillow, NumPy
